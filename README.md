Чат на веб сокетах. 

## Используемый стек

- node.js v23.0.0
- ts

## Установка
- создайте свой файл env
- npm install
- npx tsc
- node app_ts.js

## Реализован функционал: 

- авторизация по JWT токенам
- чат в реал тайме
- защита от спама
- возможность получать и отправлять данные в другие сервисы
